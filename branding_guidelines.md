| Color        | HEX         | Suggested Use                           |
| ------------ | ----------- | --------------------------------------- |
| Deep Navy    | **#0D1B2A** | Titles, logo, headers, dark backgrounds |
| Primary Blue | **#1E6BD6** | Buttons, highlights, charts, links      |
| Light Blue   | **#6FAEF7** | Secondary charts, accents, hover states |
| Teal         | **#00B4B4** | KPIs, positive metrics, callouts        |
| Light Gray   | **#E6E9EF** | Dashboard backgrounds, cards, dividers  |
| Slate Gray   | **#4A5568** | Body text, axes, labels, secondary text |

### Dashboard Theme

**Background**

* General: `#FFFFFF` (White)
* Cards or panels: `#E6E9EF` (Light Gray)

**Text**

* Headings: `#0D1B2A` (Deep Navy)
* Body: `#4A5568` (Slate Gray)

**Charts**

* Primary series: `#1E6BD6` (Primary Blue)
* Secondary series: `#6FAEF7` (Light Blue)
* Highlight or KPI: `#00B4B4` (Teal)
